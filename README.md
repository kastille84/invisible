# Invisible
This is a small app that gets the weather and current time of a give location

### Setup
```bash
# install application dependencies
npm install # or `npm i`

# to run the compiled code (you can stop here)
npm run app

# to develop in typescript you require a TypeScript compiler (tsc) either globally or locally
 npm install -g typescript

# run the application for development
npm run start

# run test
npm run test
```

### View Application
Application is CLI based. The Output is console.logged on to the terminal.
